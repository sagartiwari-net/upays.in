# UPIPays aMember Pro Plugin

The aMember plugin source lives at:

`payment-hub/internal/assets/amember/upipays.php`

## Install

1. Super admin: **Admin → Downloads → aMember plugin** (ZIP)
2. Or merchant dashboard → **Integrations**
3. Extract `upipays.php` to `application/default/plugins/payment/` on your aMember site
4. Enable plugin in aMember admin → Setup → Plugins

Docs: https://upays.in/docs/plugins/amember
