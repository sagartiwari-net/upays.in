Verify Order

GET /api/v1/orders/{order_id}/verify

Poll payment status using your order_id (the ID you sent when creating the order). Use as backup if webhook delivery fails.

Example

GET https://upays.in/api/v1/orders/INV-1001/verify
X-Merchant-Key: mk_xxx
X-Timestamp: 1717654321
X-Signature: ... (sign with empty body)

Success response (200)

{
  "success": true,
  "data": {
    "hub_order_id": "HUB-000042",
    "order_id": "INV-1001",
    "status": "success",
    "amount": 499.00,
    "currency": "INR",
    "paid_at": "2026-06-06T12:15:00Z"
  }
}
