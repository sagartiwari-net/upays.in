Getting Started

Accept UPI payments on your website in under 30 minutes with UPIPays dynamic QR + email verification.

Overview

UPIPays is a payment hub. Your server creates an order via REST API, customer pays via UPI QR on our hosted checkout page, we verify payment via bank email (IMAP), then notify your site via webhook.

  • 0% transaction fee — money goes directly to your bank
  • HMAC-signed API — secure server-to-server calls
  • Hosted checkout — https://upays.in/pay/{token}

Quick start

  1. Register as a merchant
  2. Complete UPI onboarding (UPI ID + Gmail IMAP app password)
  3. Copy API Key and API Secret from Settings
  4. Set your default Webhook URL in dashboard or per-order
  5. Call POST /api/v1/orders/create — redirect customer to payment_url
  6. Listen for payment.success webhook or poll GET /api/v1/orders/:id/verify
Base URL: https://upays.in
All API paths are prefixed with /api/v1.

Test payment

Use amount ₹1 for testing. Each order gets a unique pay amount (e.g. ₹1.03) so bank emails can be matched automatically.
