aMember Pro Plugin

Official UPIPays payment plugin for aMember Pro membership sites.

Download

Download from super admin: Admin → Downloads → aMember plugin, or API:

GET https://upays.in/admin/api/downloads/amember-plugin
Authorization: Bearer {admin_jwt}

File: upipays.php — place in your aMember application/default/plugins/payment/ folder.

Configuration

  1. aMember Admin → Setup → Plugins → enable upipays
  2. Add payment system → UPIPays
  3. Set Hub URL: https://upays.in
  4. Paste API Key and API Secret from UPIPays merchant dashboard
  5. Save — webhook URL is auto-configured to your aMember IPN endpoint

Test

  1. Create a ₹1 test product in aMember
  2. Complete checkout — you'll be redirected to UPIPays QR page
  3. Pay via UPI — membership activates on webhook
Regenerating API secret in UPIPays dashboard requires updating aMember plugin settings too.
