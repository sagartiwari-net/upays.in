WooCommerce Plugin

Accept UPI on WordPress / WooCommerce stores.

Download

Merchant dashboard → Integrations → Download WooCommerce plugin

Install

  1. WordPress Admin → Plugins → Add New → Upload Plugin
  2. Upload upipays-woocommerce.zip and activate
  3. WooCommerce → Settings → Payments → UPIPays UPI
  4. Hub URL: https://upays.in
  5. API Key + Secret from dashboard → Settings

Webhook

https://yoursite.com/?wc-api=wc_gateway_upipays

Sent automatically with each order — marks WooCommerce order paid on payment.success.

Test

Create a ₹1 product, checkout with UPI, pay on QR page. Saving gateway settings runs a ₹1 API connectivity test.
