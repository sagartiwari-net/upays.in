Create Order

POST /api/v1/orders/create

Request body

FieldTypeRequiredDescription
order_idstringYesYour unique order reference (max 100 chars)
amountnumberYesOrder amount in INR (must be > 0)
currencystringNoDefault INR (only INR supported)
customer.emailstringNoCustomer email
customer.namestringNoCustomer name
product.namestringNoProduct label on checkout
return_urlstringYesRedirect URL after payment attempt
webhook_urlstringNoOverride merchant default webhook

Example request

POST https://upays.in/api/v1/orders/create
X-Merchant-Key: mk_xxx
X-Timestamp: 1717654321
X-Signature: abc123...
Content-Type: application/json

{
  "order_id": "INV-1001",
  "amount": 499.00,
  "currency": "INR",
  "customer": { "email": "user@example.com", "name": "Rahul" },
  "product": { "name": "Premium Plan" },
  "return_url": "https://yoursite.com/payment/return",
  "webhook_url": "https://yoursite.com/payment/webhook"
}

Success response (201)

{
  "success": true,
  "data": {
    "hub_order_id": "HUB-000042",
    "payment_url": "https://upays.in/pay/abc123token",
    "expires_at": "2026-06-06T12:30:00Z"
  }
}

Redirect the customer to payment_url to complete UPI payment.

Errors

See Error Codes for full list. Common: 409 duplicate order_id, 402 plan limit exceeded.
